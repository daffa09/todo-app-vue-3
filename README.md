### SIMPLE TODO APP WITH VUE 3

this is a repo for a simple todo app with vue 3

## SYSTEM REQUIREMENTS

- node v18.0.0
- git bash for running commands

## GETTING STARTED

- Clone the repo
- run

```
npm install
```

or you can use yarn

```
yarn install
```

- run

```
npm run dev
```

or if you are using yarn

```
yarn dev
```
